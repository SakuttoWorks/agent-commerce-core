# Legal Disclaimer & Algorithm Scope

**Last Updated: March 8, 2026**

## 1. Scope of Codebase
The source code contained in this repository (`agent-commerce-core`) is strictly defined as a **Data Normalization Engine** for Project GHOST SHIP.
Its primary function is to:
- Scrape public web data from specified URLs.
- Parse HTML/DOM structures.
- Convert unstructured text into machine-readable formats (JSON/Markdown).

## 2. No Predictive Algorithms
This repository **does not contain** any logic, models, or algorithms designed for:
- Financial market prediction or forecasting.
- Automated trading execution.
- Investment advice generation.
- Alpha generation or signal processing.
